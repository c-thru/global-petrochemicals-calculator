2. Upstream processes#

These processes produce the basic input chemicals such as syngas, ethyl alcohol, methyl alcohol, hydrogen, used subsequently in the Primary chemicals processes.

Most of the main chemicals are already defined from the UK FIRES production system map [Lupton et al., 2024], but a few additional “objects” need to be defined here.

Ethyl alcohol, an intermediate for ethylene production, is produced from biomass by fermentation.

Object: EthylAlcohol
